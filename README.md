## 🇨🇳  学习方式
教程以 `git` 分支的方式管理

<br/>

1. 通过 `git` 把项目复制到本地
```git
git clone https://github.com/ikcamp/koa2-tutorial
```

2. 切换目录
```
cd koa2-tutorial
```

3. 在当前目录下切换分支
```
git checkout 0-start
```

**注意：** 所有的分支命名上，都是数字开头，也就是我们的开发顺序和讲解顺序。
